# Addressable Balancer LEDs

Addressable LEDs that connect through a battery's balancer plug. The project combines compact FPV-friendly electronics with individually controllable lighting.

[View the project on GitHub](https://github.com/KidCe/adressable-balancer-led){ .md-button .md-button--primary }

## Project scope

- Hardware and wiring for balancer-connected LEDs
- Addressable lighting control
- Source files and ongoing documentation in the GitHub repository
